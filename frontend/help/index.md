# Willkommen beim Maintenance Tool

Diese Hilfeseite unterstützt Sie bei der Planung von Wartungsfenstern, der Verwaltung von Anwendungen und dem automatischen Versand von Benachrichtigungs-E-Mails.

## Erste Schritte

Wählen Sie eine Sektion aus der Seitenleiste:

- **Dashboard:** Erste Übersicht der gehosteten Web-Applikationen und geplanten Termine.
- **Administration:** Anleitung für Administratoren zur Rollenvergabe.

Bei weiteren Fragen besuchen Sie unsere [FAQ-Seite](./faq.md).
