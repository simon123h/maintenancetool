# Häufig gestellte Fragen (FAQ)

### Wer darf Wartungsfenster erstellen?
Benutzer mit den Rollen `ROLE_MANAGER` oder `ROLE_ADMIN` dürfen geplante Wartungen eintragen, Anwendungen registrieren und E-Mail-Vorlagen definieren.

### Wie werden Benachrichtigungs-E-Mails verschickt?
Wenn ein Wartungsfenster angekündigt wird, liest das System die hinterlegte Liste der betroffenen Anwendungsbenutzer aus und versendet automatisch E-Mails basierend auf dem ausgewählten Template.
