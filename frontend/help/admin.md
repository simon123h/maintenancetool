# Anleitung für Administratoren

Als Administrator (`ROLE_ADMIN`) haben Sie vollen Zugriff auf das System.

## Funktionen
- **Benutzerberechtigungen:** Im Administrationsbereich können Sie Berechtigungen an andere Konten vergeben.
- **Rollen:** Vergeben Sie administrative Rollen wie `ROLE_ADMIN` oder `ROLE_MANAGER` für die Verwaltung von Anwendungen und Wartungsfenstern.
